# Examples

This folder contains example documents demonstrating the Word to IEEE converter.

## Files

- **test_document.docx** - Sample Word document with typical academic paper structure (title, abstract, sections, subsections, references)
- **test_document_IEEE.docx** - The same document after conversion to IEEE format

## Usage

To test the converter with these examples:

```bash
python word_to_ieee.py examples/test_document.docx
```

This will create a new IEEE-formatted version of the document.
